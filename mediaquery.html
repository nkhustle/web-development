Within web development there are many different jobs which require different training to have been taken before you would be classified as a professional. These are jobs such as web software engineer, web developer, application developer and a front end developer- optimisation and personalisation. When comparing the jobs the difference is that certain requirements such as making a application require a different approach compared to web software engineering because you will not use the same software to do this for example when developing an application you will use software such as Jira, out systems and data dog cloud monitoring but when developing web software your most likely to be using platforms such as java and Microsoft share point.
