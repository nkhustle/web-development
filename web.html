Web Development Glossery

301 redirect – A permanent redirect from one URL to another, usually from your old website to the new website

404 – A page a user sees when they try to reach a non-existent page on your website.

ALT attribute – Used to specify the alternate text that is displayed inside the image placeholder while the page is loading.

Breakpoints – The points at which a website's content will adjust to provide the user with the best possible layout to view content.

Call to Action (CTA) – Specific text, image, banner or button that uses persuasive, action-oriented words that urges a visitor on a website to act.

Cookies – A small text file sent which includes anonymous unique identifier and visit information that is sent to a browser from a website and stored on a visitor's computer hard drive.

Content Management System (CMS) – A software system that is used to control the content on your website. This allows you to login into the "backend" of your website and edit the text and images.

Conversion – When a user takes a specific desired action related to online marketing and lead generation.

CSS – Cascading Style Sheets (CSS) are the code that developers use to designate how a web page should be presented to the end user.

Domain – The name of the website that people type into a browser to visit it. For example, our domain is circlesstudio.com.

Favicon – A small icon image that is associated with a specific website, usually containing the company logo or a variation.

Hosting – The web servers where your website files are housed, served, and maintained.

HTML – Hypertext Markup Language (HTML) is the markup language that is used to construct web pages and display content like text, images, video and links on the web.

Information Architecture (IA) – The information design, organization of content and navigation of a website.

Meta Tag – An HTML tag that stores information about a web page, such as description, author, copyright, etc.

Page template – A unique page layout for a website, specifically websites that are built using content management systems.

Registrar – The company used to register your domain name. Some examples include GoDaddy or Network Solutions.

Responsive design – A website that adjusts to the screen it is being viewed on, whether desktop, mobile or smart phone.

Search Engine Optimization (SEO) – Helping search engines understand the information on your website in order to rank higher in organic search results.

Slider – A rotating banner of images that is usually placed on the homepage of a website. It is a "slide show" type format that can highlight different content and include images or video.

Site map – A document that shows a global, hierarchical view of a website's pages and content.

User experience – The interaction a user has with an interface. From a planning perspective, the user experience is typically defined in wireframes.

Wireframe – A visual guide to show the content of a web page without any design elements.

WYSIWYG – Stands for "What You See Is What You Get."
